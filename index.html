🎱 Magic 8 Ball - Игра-предсказатель

Описание: Python-реализация классической игрушки Magic 8 Ball. Пользователь задаёт вопрос, программа даёт случайный ответ из предопределённого списка.

Python Random Module User Input

Функционал:

  • Принимает вопрос от пользователя
  • Генерирует случайный ответ из 9 вариантов
  • Использует условную логику (if-elif-else)
import random
question = input('question: ')
num = random.randint(1, 9)
if num == 1:
    answer = 'Yes - definitely.'
elif num == 2:
    answer = 'It is decidedly so.'
# ... и другие варианты
print('Magic 8 Ball: ' + answer)

Чему я научилась:

  • Работа с модулем random
  • Условные операторы
  • Обработка ввода пользователя
  • Управление логикой программы

👤 Face Recognition - Система распознавания лиц

Описание: Проект в области компьютерного зрения с использованием OpenCV и библиотеки face_recognition. Программа открывает доступ к веб-камере и отслеживает лицо пользователя в реальном времени.

Python OpenCV Face Recognition Tkinter (GUI) PIL

Функционал:

  • Захват видеопотока с веб-камеры
  • Отображение в реальном времени через GUI
  • Кнопки "Login" и "Register New User" для будущих функций
  • Обработка изображений через OpenCV

Технические детали:

# Инициализация камеры
self.cap = cv2.VideoCapture(0)

# Обработка кадров
ret, frame = self.cap.read()
img_ = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
self.most_recent_capture_pil = Image.fromarray(img_)

Текущий статус: В разработке. Базовая функциональность работает, но алгоритм распознавания требует дополнительной настройки для повышения точности.

Чему я научилась:

  • Работа с видеопотоком через OpenCV
  • Основы компьютерного зрения
  • Создание GUI с помощью Tkinter
  • Подключение внешних библиотек Python
  • Отладка и настройка алгоритмов

Планы по улучшению:

  • Реализовать функцию распознавания известных лиц
  • Добавить систему логирования входов
  • Улучшить точность детекции при разном освещении
  • Создать базу данных для хранения данных пользователей
